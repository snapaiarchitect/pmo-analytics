# Data Sources

## Primary Data Sources

| Source | Type | Description | URL |
|--------|------|-------------|-----|
| USASpending.gov | Government API | Federal contracts, grants, and awards | https://www.usaspending.gov/ |
| GAO Recommendations | Government Database | Government Accountability Office open recommendations | https://www.gao.gov/ |
| IT Dashboard | Government API | Federal IT project cost and schedule performance | https://itdashboard.gov/ |
| Federal Project Data | Government Dataset | Federal project risk and schedule data | https://www.itdashboard.gov/ |

## Data Provenance

- USASpending API for federal award data by agency and program
- GAO for project risk and recommendation tracking
- IT Dashboard for technology project performance metrics

## Data Files

No local CSV — data fetched on-demand via API or downloaded from public sources.

## Refresh Strategy

- USASpending updates daily
- GAO updates continuously
- IT Dashboard refreshes quarterly
- Re-fetch via APIs for latest data
